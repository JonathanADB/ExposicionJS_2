DOM-Atributos cambios y modificaciones

En JavaScript, los atributos DOM (Document Object Model) son propiedades de elementos HTML a los que se puede acceder y manipular usando JavaScript.

Los atributos en el DOM se pueden clasificar ampliamente en dos tipos

    Atributos estándar

  • Los atributos estándar son atributos predefinidos que son comunes a la mayoría de los elementos HTML
  • Ejemplos incluyen id, class, style, src, href, etc
  • Proporcionan información adicional o ajustes para el elemento HTML correspondiente.

    • Atributos no estándar

  • A menudo se utilizan para almacenar datos o información adicional relacionada con un elemento, y su comportamiento es definido por el desarrollador
  • son atributos definidos por el usuario que no forman parte del estándar HTML

Todos los atributos son accesibles usando los siguientes métodos:

  • elem.hasAttribute(nombre) -- comprueba si existe.
  • elem.getAttribute(nombre) -- obtiene el valor.
  • elem.setAttribute(nombre) -- establece el valor.
  • elem.removeAttribute(nombre) -- elimina el atributo.

DOM-CSS Cambios y modificaciones

Mientras que el CSS en sí no se manipula directamente usando JavaScript, el Document Object Model (DOM) se puede usar para interactuar y modificar las propiedades CSS dinámicamente.

Aquí hay una breve descripción de cómo se puede trabajar con CSS en el DOM usando JavaScript

1. Acceso a elementos

Puede usar JavaScript para acceder a elementos HTML en el DOM. Esto se suele hacer utilizando métodos como:

  • `getElementById`
  • `getElementsByClassName`
  • `getElementsByTagName`
  • `querySelector`

2. Modificar propiedades CSS

Una vez que tenga una referencia a un elemento HTML, puede modificar sus propiedades CSS usando la propiedad:

  • `style`

3. Añadir/eliminar clases en CSS

También se puede agregar o quitar clases CSS desde un elemento usando el:

  • `classList`

4. Configuración de texto en CSS

Si deseas configurar varias propiedades CSS a la vez, puedes utilizar el:

  • `cssText`

5. obtener Computed Styles

Puede recuperar los estilos computados de un elemento usando el método:

  • `getComputedStyle`

6. Manejo de eventos

Puedes usar JavaScript para cambiar dinámicamente los estilos en respuesta a las acciones del usuario u otros eventos
